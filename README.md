# 2048

### running the game

```bash
  $ git clone https://github.com/glucero/2048
  $ cd 2048
  $ rake start
```

### quick play

```bash
  $ curl -s0Lo- http://git.io/M6K_0Q | ruby
```

### instructions

Use the arrow keys to shift the tiles up, down, left and right and try to create a 2048 tile.
